applyTerminalAction

protected open fun applyTerminalAction(demand: SupplyChainModel.Demand)(source)

Hook fired when service ends for demand. Default invokes the configured terminalAction — calling demand.store() or routing + re-ship per DockTerminalAction.Ship.

Subclasses overriding this should call demand.store() or demand.ship() themselves to finalise the demand's lifecycle.
