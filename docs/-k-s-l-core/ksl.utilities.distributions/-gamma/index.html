Gamma

class Gamma(theShape: Double = 1.0, theScale: Double = 1.0, name: String? = null) : Distribution<Gamma> , ContinuousDistributionIfc, LossFunctionDistributionIfc, InverseCDFIfc, GetRVariableIfc

Models random variables that have gamma distribution For more information on the gamma distribution and its related functions, see "Object-Oriented Numerical Methods" by D. Besset
