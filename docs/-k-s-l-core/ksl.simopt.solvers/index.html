MooreNeighborhoodFinder
Link copied to clipboard
class MooreNeighborhoodFinder(val problemDefinition: ProblemDefinition, radius: Int = 1) : NeighborhoodFinderIfc

A Moore neighborhood includes all points where the Chebyshev distance (also called L∞ distance) from the center is less than or equal to the radius. The Chebyshev distance is the maximum absolute difference across all dimensions. This function uses a direct iteration around a bounding box. By default, this implementation does not include the center point of the neighborhood and uses a radius of 1.

Solver

abstract class Solver(val problemDefinition: ProblemDefinition, evaluator: EvaluatorIfc, maximumIterations: Int, var replicationsPerEvaluation: ReplicationPerEvaluationIfc, name: String? = null) : IdentityIfc, Comparator<Solution> , IterationEmitterIfc, LifeCycleEmitterIfc

A solver is an iterative algorithm that searches for the optimal solution to a defined problem. In this abstract base class, the algorithm is conceptualized as having a main iterative loop. The loop is the main loop that ultimately determines the convergence of the algorithm and recommended solution. Some algorithms have "inner" loops". In general, inner loops are used to control localized search for solutions. If an algorithm has additional inner loops, these can be embedded within the main loop via the subclassing process.

class VonNeumannNeighborhoodFinder(val problemDefinition: ProblemDefinition, radius: Int = 1) : NeighborhoodFinderIfc

A von Neumann neighborhood includes all points where the Manhattan distance (sum of absolute differences in all dimensions) from the center is less than or equal to the radius. This function uses a direct iteration around a bounding box. By default, this implementation does not include the center point of the neighborhood and uses a radius of one.
